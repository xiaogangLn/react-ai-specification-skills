# Coding Style Rules

---

# TypeScript

Always use TypeScript.

Avoid using any.

---

# Import Order

1 React  
2 Third party libraries  
3 Internal packages  
4 Local modules

---

# File Naming

Use kebab-case.

Example:

user-table.tsx

---

# Component Naming

Use PascalCase.

Example:

UserTable

---

# Hook Naming

Hooks must start with "use".

Example:

useUserList
